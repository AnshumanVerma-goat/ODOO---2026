# React + TypeScript + Vite

This is the frontend for the Fleet Management hackathon project.

## Roles

| Role | Focus Areas |
|------|-------------|
| **Fleet Manager** | Fleet, Maintenance, vehicle lifecycle |
| **Driver** | Trips, active deliveries |
| **Safety Officer** | Drivers, compliance, safety scores |
| **Financial Analyst** | Fuel & Expenses, Analytics |

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173 — use the **View as** dropdown to switch between roles.

## Pages

- Dashboard (role-specific overview)
- Fleet
- Drivers
- Trips
- Maintenance
- Fuel & Expenses
- Analytics
- Settings

All data is mocked for demo purposes — ready to wire up to a backend API.
